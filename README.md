# Vasicek Interest Rate Model

This project studies the Vasicek short-rate model and its applications in quantitative finance.

The notebook combines theoretical derivations, stochastic calculus, and numerical simulations to analyze the dynamics of interest rates and zero-coupon bond pricing.

Main topics covered:

- stochastic differential equations (SDEs),
- Ornstein–Uhlenbeck processes,
- Monte Carlo simulation,
- zero-coupon bond pricing,
- yield curve dynamics,
- analytical properties of the Vasicek model,
- numerical methods for short-rate models.

The project also includes graphical visualizations of simulated interest-rate trajectories and bond price dynamics.

## Technologies

- Python
- NumPy
- SciPy
- Matplotlib
